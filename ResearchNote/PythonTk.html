Python tkinter

tkinter為Python原生的視窗應用程式套件，所以只要有原本的Python環境即可開發，不用下載其他的library。 雖然需要透過程式碼編寫介面，但還算易於使用，適合初學者開發功能較單純的使用介面。

範例實作

視窗基礎設定

首先將tkinter的模組引用(import)進來，並把他簡稱為tk

import tkinter as tk

建立一個tkinter物件，並且加入維持視窗運行的method

win = tk.Tk() 
win.mainloop() # 維持視窗顯示的迴圈，沒有這行程式碼視窗無法維持顯示

此時應該可以叫出一個空白視窗出來

呼叫title的方法去設定視窗最上方的標題名稱

win.title("Python tkinter")

也可以更改視窗左上的圖標，ico檔必須放在同目錄底下

win.iconbitmap("PythonIcon.ico")

調整視窗大小

win.geometry("400x300")#400是橫軸長度; 300是縱軸長度

更改視窗的背景顏色，透過config去更改win視窗內的屬性

win.config(bg="skyblue") # 更改bg屬性，顏色可以用字典key或16進位hex code指定

視窗文字設定

要在視窗裡放置文字，須建立一個tkinter裡的Label物件。

在創立Label物件時，會有幾個常見的屬性可以調整。

  1. 第一個參數為label要放置的容器名稱，目前只有主視窗win。tkinter還有其他容器物件如: Frame, Canvas
  2. text為label欲顯視的文字內容
  3. font為字體的設定(字體樣式, 字體大小, 粗體字)
  4. bg可調整label背景顏色
label1 = tk.Label(win, # Label要顯示的視窗
                  text="Python tkinter", # label的文字內容
                  font=('Microsoft JhengHei UI', 20, 'bold'), # 設定字體樣式
                  bg="skyblue" # 背景顏色)

label放置位置，tkinter元件的放置位置主要有三種方法: pack, grid, place

這邊先說明place的使用方式，place是透過絕對座標(x,y)來指定元件位置。用法如下: 

label_1.place(x=80, y=100) # 放置位置
